# Diverga v6.7.0: Category I - Systematic Review Automation

**Release Date**: 2026-01-30

## Overview

Diverga v6.7.0 introduces **Category I: Systematic Review Automation** with 4 new agents (I0-I3) for PRISMA 2020 compliant literature review pipelines. This release expands Diverga from 40 to 44 agents across 9 categories.

---

## New Agents

### Category I: Systematic Review Automation

| Agent ID | Name | Model | Tier | Purpose |
|----------|------|-------|------|---------|
| **I0** | ScholarAgentOrchestrator | Opus | HIGH | Pipeline coordination, stage management |
| **I1** | PaperRetrievalAgent | Sonnet | MEDIUM | Multi-database fetching (Semantic Scholar, OpenAlex, arXiv) |
| **I2** | ScreeningAssistant | Sonnet | MEDIUM | AI-PRISMA 6-dimension screening |
| **I3** | RAGBuilder | Haiku | LOW | Vector database construction (zero cost) |

## Pipeline Stages

The systematic review automation follows a 7-stage PRISMA 2020 compliant pipeline:

```
Stage 1: Research Domain Setup      ‚Üí config.yaml, project initialization
Stage 2: Query Strategy             ‚Üí Boolean search strings, database selection
Stage 3: Paper Retrieval            ‚Üí I1-paper-retrieval-agent
Stage 4: Deduplication              ‚Üí 02_deduplicate.py
Stage 5: PRISMA Screening           ‚Üí I2-screening-assistant (Groq LLM)
Stage 6: PDF Download + RAG         ‚Üí I3-rag-builder
Stage 7: Documentation              ‚Üí PRISMA diagram generation
```

## Project Types

I0 orchestrator supports two project types with different screening thresholds:

### knowledge_repository
- **Threshold**: 50% confidence (score ‚â• 25)
- **Expected output**: 5,000-15,000 papers
- **Use case**: Teaching materials, AI research assistant, domain exploration

### systematic_review
- **Threshold**: 90% confidence (score ‚â• 40)
- **Expected output**: 50-300 papers
- **Use case**: Meta-analysis, journal publication, clinical guidelines

## Usage Examples

### Claude Code (Task Tool)

```python
# Invoke I0 orchestrator
Task(
    subagent_type="diverga:i0",
    model="opus",
    prompt="I want to conduct a PRISMA 2020 systematic review on AI chatbots in language learning"
)

# Direct paper retrieval
Task(
    subagent_type="diverga:i1",
    model="sonnet",
    prompt="Fetch papers from Semantic Scholar, OpenAlex, and arXiv for query: AI chatbots language learning"
)
```
