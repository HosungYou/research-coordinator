## Overview / ê°œìš”

### English

Diverga orchestrates **37 specialized research agents** across 8 categories to support the complete research lifecycle. Unlike single-agent systems, Diverga uses intelligent routing, parallel execution, and sequential chaining to maximize efficiency and quality.

**Core Principles:**
- **Human-Centered**: AI assists, humans decide at critical checkpoints
- **VS Methodology**: Prevents mode collapse with creative alternatives
- **Parallel Execution**: Multiple agents work simultaneously when possible
- **Context Persistence**: Research context maintained across entire project

## Conclusion / ê²°ë¡ 

### English

Diverga's 37-agent architecture enables sophisticated research workflows through:

1. **Intelligent Routing**: Right agent for right task
2. **Parallel Execution**: Multiple agents work simultaneously
3. **Sequential Chaining**: Output flows between agents
4. **Human Checkpoints**: Critical decisions remain with researchers
5. **B5 Innovation**: Large-scale document processing without memory overflow

The key to effective orchestration is understanding when to use sequential vs parallel execution, and leveraging B5 for document-heavy workflows.
