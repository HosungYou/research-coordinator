## Overview

Diverga v6.0 represents a fundamental shift in design philosophy: from autonomous AI operation to **human-centered collaboration**. This release removes all mechanisms that could bypass human decision-making, ensuring that strategic research choices always remain with the researcher.

## What Was Removed

### Sisyphus Protocol âŒ

- **Was**: "Work never stops until complete"
- **Problem**: Could bypass human checkpoints, enabling autonomous operation without approval
- **Now**: AI stops at EVERY checkpoint and waits for human decision

### Iron Law of Continuation âŒ

- **Was**: "Move to next agent OR human checkpoint"
- **Problem**: The "OR" made checkpoints optional, allowing AI to skip them
- **Now**: Sequential flow: "checkpoint THEN next agent"

### OMC Autonomous Modes âŒ

| Removed Mode | Problem |
|--------------|---------|
| `ralph` | Persistence mode that could bypass checkpoints |
| `ultrawork` | Maximum parallelism without human oversight |
| `autopilot` | Full autonomous execution |
| `ecomode` | Token-efficient mode that skipped confirmations |
| `swarm` | Multi-agent coordination without checkpoints |
| `ultrapilot` | Parallel autopilot without approval |

**Kept**: Model routing (Opus/Sonnet/Haiku tiers) - this only affects cost/performance, not human oversight

### Legacy Folder Structure âŒ

- Removed: `.omc/` folder and all contents
- Migrated: Necessary configuration moved to `.claude/`

## Migration Guide

### From v5.0 (Sisyphus Edition)

1. **Delete `.omc/` folder** (if it exists)
   ```bash
   rm -rf .omc/
   ```

2. **Update your workflow expectations**:
   - AI will now STOP at checkpoints and WAIT for your approval
   - No more autonomous "ralph" or "ultrawork" modes
   - You must explicitly approve research direction, theory selection, and methodology

3. **Model routing still works**:
   - Use `haiku` for simple tasks (effect size extraction, code generation)
   - Use `sonnet` for standard tasks (literature search, quality appraisal)
   - Use `opus` for strategic tasks (theory selection, analysis design)

### Breaking Changes

| Feature | v5.0 | v6.0 |
|---------|------|------|
| Checkpoints | Optional (could be bypassed) | Mandatory |
| OMC modes | ralph, ultrawork, ecomode available | Removed |
| Agent naming | Numbered (01-21) | Category-based (A1-H2) |
| State location | `.omc/` | `.claude/` |
| Agent count | 27 | 33 |
