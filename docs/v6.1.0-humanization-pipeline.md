## 1. New Agents

### 1.1 G5-AcademicStyleAuditor

**Purpose**: Detect AI-generated patterns in academic writing
**Model Tier**: MEDIUM (Sonnet)
**Category**: G - Publication & Communication

**Capabilities**:
- AI pattern detection across 24 categories
- AI probability scoring (0-100%)
- Pattern density analysis
- Risk classification (high/medium/low)
- Section-specific detection (abstract, methods, discussion, etc.)
- Vocabulary tier analysis

**Input/Output**:
```yaml
Input:
  required:
    - text: "Academic text to analyze"
  optional:
    - section_type: "abstract|methods|results|discussion"
    - analysis_depth: "quick|standard|deep"
    - sensitivity: "low|medium|high"

Output:
  - ai_probability: 67%
  - pattern_count: 18
  - patterns_by_category: {content: 5, language: 8, style: 3, ...}
  - patterns_by_risk: {high: 5, medium: 9, low: 4}
  - recommendations: ["Address L1 vocabulary", "Vary transitions"]
```

### 2.2 Vocabulary Tiers

**Tier 1 - High Alert** (Always flag):
```yaml
- delve, dive into, crucial, foster, intricate, intricacies
- realm, landscape, multifaceted, comprehensive, underscore
- noteworthy, meticulous, leverage, utilize, pivotal
- embark, endeavor, unveil, unravel, intriguing
```

**Tier 2 - Moderate Alert** (Flag if clustered):
```yaml
- robust, streamline, facilitate, enhance, fundamental
- substantial, significant, paramount, innovative
- nuanced, sophisticated, compelling, profound
```

**Tier 3 - Context Check** (Evaluate in context):
```yaml
- important, effective, relevant, appropriate
- demonstrate, indicate, suggest, reveal
```

## 3. Transformation System

### 3.1 Transformation Modes

#### Conservative Mode
- **Target**: HIGH-RISK patterns only
- **AI Probability Reduction**: 20-35%
- **Text Change**: 5-15%
- **Best For**: Journal submissions, formal publications
- **Preserves**: Most original phrasing

```yaml
patterns_targeted:
  - C1, C4, C5 (Content)
  - L1_tier1 (Language)
  - S5 (Style)
  - M1, M2 (Communication)
  - A1, A6 (Academic)
```

#### Balanced Mode (Recommended)
- **Target**: HIGH + MEDIUM-RISK patterns
- **AI Probability Reduction**: 35-50%
- **Text Change**: 15-30%
- **Best For**: Most academic writing
- **Preserves**: Technical accuracy

```yaml
patterns_targeted:
  - All HIGH-RISK
  - C2, C3 (Content)
  - L2, L4, L6 (Language)
  - S1, S3 (Style)
  - H1, H2 (Hedging)
  - A4, A5 (Academic)
```

#### Aggressive Mode
- **Target**: ALL patterns
- **AI Probability Reduction**: 50-70%
- **Text Change**: 30-50%
- **Best For**: Blog posts, informal writing
- **Warning**: May affect scholarly tone

```yaml
patterns_targeted:
  - All patterns (C1-C6, L1-L6, S1-S6, M1-M3, H1-H3, A1-A6)
```

### 3.2 Transformation Examples

**L1 - AI Vocabulary**:
```
Before: "This study delves into the crucial role of..."
After:  "This study examines the important role of..."

Before: "We leverage these findings to foster..."
After:  "We use these findings to support..."
```

**C1 - Significance Inflation**:
```
Before: "This groundbreaking research revolutionizes..."
After:  "This research advances understanding of..."

Before: "A paradigm shift in educational technology..."
After:  "A notable development in educational technology..."
```

**H2 - Excessive Hedging**:
```
Before: "This could potentially perhaps suggest..."
After:  "This may suggest..."

Before: "It might be possible that there is a tendency..."
After:  "There appears to be a tendency..."
```

### 3.3 Preservation Rules

**NEVER Transform**:
```yaml
absolute_preservation:
  - Citations: "(Author, year)" format
  - Statistics: p-values, effect sizes, CIs, N values
  - Direct quotes: Verbatim quoted text
  - Technical terms: Field-specific terminology
  - Mathematical expressions: Formulas, equations
  - Proper nouns: Names, places, organizations
  - Methodology terms: "randomized controlled trial", etc.
```

**Context-Sensitive Preservation**:
```yaml
section_specific:
  methods:
    - Preserve all procedural language
    - Keep technical descriptions exact
  results:
    - Preserve all numerical values
    - Keep statistical reporting format
  discussion:
    - Allow more transformation
    - Maintain scholarly hedging
```

## 7. Ethics Framework

### 7.1 Core Principle

> **"Humanization improves expression, not deception."**

### 7.2 What Humanization Does vs. Does NOT Do

**DOES**:
- Help express ideas naturally
- Remove robotic phrasing
- Maintain scholarly tone while improving readability
- Remove obvious AI artifacts

**DOES NOT**:
- Make AI use "undetectable"
- Replace need for AI disclosure
- Generate original ideas
- Substitute for human judgment
- Guarantee acceptance

### 7.3 Researcher Responsibilities

```yaml
before_using:
  - Check institution's AI policy
  - Check target journal's AI requirements
  - Understand funding agency requirements
  - Know disciplinary norms

maintain:
  - Intellectual ownership of research
  - Original research question formulation
  - Methodology design/selection
  - Data collection and analysis
  - Interpretation and conclusions
```
