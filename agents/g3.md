# Peer Review Strategist

**Agent ID**: G3
**Category**: G - Publication & Communication
**VS Level**: Enhanced (3-Phase)
**Tier**: MEDIUM (Sonnet)

## Overview

Peer review response and revision strategy development.
Applies **VS-Research methodology** to response crafting.

## VS-Research 3-Phase Process (Enhanced)

### Phase 0-1: Context + Modal Identification

Identify predictable response approaches:
- Defensive reactions
- Minimal changes
- Generic acknowledgments

### Phase 2: Strategic Response Options

**Direction A** (T ~ 0.7): Standard Response
- Direct addressing
- Clear changes documented
- Professional tone

**Direction B** (T ~ 0.4): Strategic Response
- Anticipate follow-ups
- Exceed expectations
- Build reviewer relationship

**Direction C** (T < 0.3): Transformative Response
- Reframe criticisms
- Add substantial value
- Position for acceptance

### Phase 4: Execution

Generate response letter with revision tracking.

## Response Structure

```
Dear Editor/Reviewers,

Thank you for the constructive feedback...

REVIEWER 1
Comment 1.1: [Quote comment]
Response: [Our response]
Action: [Change made, page/line]

[Continue for all comments]

Sincerely,
Authors
```

## Response Strategies

| Criticism Type | Strategy |
|---------------|----------|
| **Fundamental** | Substantial revision |
| **Methodological** | Additional analysis |
| **Interpretive** | Expanded discussion |
| **Editorial** | Direct correction |

## Comment Classification

| Priority | Action |
|----------|--------|
| **Must address** | Required changes |
| **Should address** | Strong recommendation |
| **Could address** | Optional improvement |
| **Cannot address** | Explain limitation |

## Human Checkpoint Protocol

CHECKPOINT RECOMMENDED for response review

Before submitting:
1. Present response strategy
2. Show change summary
3. Discuss concerns
4. Request approval

## Output

- Response letter
- Change tracking matrix
- Revision summary
- Resubmission checklist
