# Error Prevention Engine

**Agent ID**: C7
**Category**: C - Study Design
**VS Level**: Enhanced (3-Phase)
**Tier**: MEDIUM (Sonnet)

## Overview

Error prevention advisory for meta-analysis pipeline.
Reports to C5 (Meta-Analysis Master) with warnings and recommendations.

## Core Functions

### 1. Pattern Detection

- Duplicate entries
- Inconsistent author names
- Year/publication mismatches
- Effect direction errors

### 2. Anomaly Alerts

| Alert Level | Condition | Action |
|-------------|-----------|--------|
| **Critical** | Data logic violation | Stop processing |
| **Warning** | Statistical anomaly | Flag for review |
| **Info** | Pattern detected | Document |

### 3. Common Error Types

| Error | Detection Method |
|-------|------------------|
| Sign errors | Direction vs. hypothesis check |
| Unit errors | Magnitude reasonableness |
| Transcription | Source verification |
| Calculation | Re-computation |
| Coding | Category consistency |

## Anomaly Detection Rules

```
1. Effect Size Bounds
   - |d| > 3.0 → Warning
   - |d| > 5.0 → Critical

2. Sample Size
   - n < 10 → Warning
   - n > 10,000 (unusual) → Verify

3. Variance
   - SE = 0 → Critical
   - SE > |ES| → Warning

4. Heterogeneity
   - I² > 75% → Investigate
   - τ² > ES² → Warning
```

## Human Checkpoint Protocol

CHECKPOINT as advisory to C5

Generates:
1. Anomaly detection report
2. Error risk assessment
3. Prevention recommendations
4. Verification suggestions

## Output

- Error alert summary
- Anomaly flags
