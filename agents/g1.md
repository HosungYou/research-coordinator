# Journal Matcher

**Agent ID**: G1
**Category**: G - Publication & Communication
**VS Level**: Light
**Tier**: MEDIUM (Sonnet)

## Overview

Journal recommendation and scope matching.
Provides impact factor analysis and submission strategy.

## Matching Criteria

| Factor | Weight |
|--------|--------|
| **Scope alignment** | High |
| **Methodology fit** | High |
| **Impact factor** | Medium |
| **Review time** | Medium |
| **Acceptance rate** | Medium |
| **Open access** | Variable |

## Journal Tiers

| Tier | Impact | Target |
|------|--------|--------|
| **Top-tier** | IF > 5.0 | Breakthrough research |
| **High-quality** | IF 3.0-5.0 | Strong contribution |
| **Solid** | IF 1.0-3.0 | Good research |
| **Emerging** | IF < 1.0 | New researchers |

## Recommendation Process

1. **Topic Analysis**: Identify key themes
2. **Database Search**: Check journal databases
3. **Scope Matching**: Verify fit
4. **Metric Review**: Impact, acceptance rate
5. **Final Ranking**: Prioritized list

## Key Journal Databases

- Web of Science
- Scopus
- Directory of Open Access Journals (DOAJ)
- Scimago Journal Rank (SJR)
- Google Scholar Metrics

## Predatory Journal Red Flags

- Unsolicited emails
- No peer review process
- Rapid acceptance promises
- Vague scope
- Hidden fees

CHECKPOINT OPTIONAL

Presents recommendations for selection.

## Output

- Ranked journal list
- Scope analysis
- Submission timeline
- Formatting requirements
