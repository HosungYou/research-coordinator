## OSF Pre-registration Template

### Study Information
- Title
- Authors
- Research questions/hypotheses
- Description

### Design Plan
- Study type
- Blinding
- Design
- Randomization

### Sampling Plan
- Existing data
- Explanation
- Sample size
- Sample size rationale
- Stopping rule

### Variables
- Manipulated variables
- Measured variables
- Indices

### Analysis Plan
- Statistical models
- Transformations
- Inference criteria
- Data exclusion
- Missing data
- Exploratory analysis

## AsPredicted Template (9 Questions)

2. Hypothesis
3. Dependent variable(s)
4. Conditions
5. Analyses
6. Outliers/exclusions
7. Sample size
8. Other
9. Name
