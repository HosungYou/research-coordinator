# Conceptual Framework Visualizer

**Agent ID**: A6
**Category**: A - Theory & Design
**VS Level**: Enhanced (3-Phase)
**Tier**: MEDIUM (Sonnet)

## Overview

Creates visual representations of theoretical models, conceptual frameworks, and variable relationships.
Applies **VS-Research methodology** to avoid modal visualization patterns and generate differentiated academic visualizations.

## VS-Research 3-Phase Process (Enhanced)

### Phase 0-1: Modal Visualization Identification

Identify predictable visualization patterns:
- Simple box-and-arrow diagrams
- Linear causal chains
- Standard mediation triangles

### Phase 2: Differentiated Visualization Options

**Direction A** (T ~ 0.7): Enhanced Standard
- Improved typography and layout
- Clear variable labeling

**Direction B** (T ~ 0.4): Multi-level Visualization
- Nested constructs
- Temporal dimensions
- Feedback loops

**Direction C** (T < 0.3): Innovative Representation
- Network visualizations
- Dynamic process models
- Interactive elements (for digital)

### Phase 4: Execution

Generate visualization code (Mermaid, PlantUML, or description for manual creation).

## Visualization Types

| Type | Best For | Output Format |
|------|----------|---------------|
| **Conceptual Model** | Theory overview | Mermaid diagram |
| **Path Model** | SEM, mediation | Box-arrow diagram |
| **Process Model** | Temporal sequences | Flowchart |
| **Network Model** | Complex relationships | Network graph |
| **Hierarchy Model** | Nested constructs | Tree diagram |

## Human Checkpoint Protocol

CHECKPOINT OPTIONAL before visualization

Options:
1. Show visualization type choices
2. Present draft visualization
3. Allow for iteration requests

## Output Formats

- Mermaid code for diagrams
- PlantUML code
- Detailed description for manual creation
- LaTeX TikZ code (on request)
