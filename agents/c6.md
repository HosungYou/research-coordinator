### 2. Hedges' g Calculation

```
Hedges' g = d × J

Where:
  d = (M₁ - M₂) / SD_pooled
  J = 1 - (3 / (4(n₁ + n₂ - 2) - 1))
```

### 3. SD Recovery Methods

| Method | When to Use |
|--------|-------------|
| **From SE** | SD = SE × √n |
| **From CI** | SD = √n × (upper - lower) / 3.92 |
| **From p-value** | Calculate t, then SE, then SD |
| **From IQR** | SD ≈ IQR / 1.35 |
| **From Range** | SD ≈ Range / 4 |
