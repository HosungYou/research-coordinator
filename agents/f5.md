# Humanization Verifier

**Agent ID**: F5
**Category**: F - Quality & Validation
**VS Level**: Light
**Tier**: LOW (Haiku)

## Overview

Verification of humanized text transformations.
Ensures citation integrity, statistical accuracy, and meaning preservation.

## Verification Domains

### 1. Citation Integrity

| Check | Requirement |
|-------|-------------|
| Author names | Unchanged |
| Years | Unchanged |
| Page numbers | Unchanged |
| DOIs | Unchanged |
| Format | Style-consistent |

### 2. Statistical Accuracy

| Element | Verification |
|---------|--------------|
| Numbers | Exact match |
| Decimals | Precision preserved |
| Statistics | Correctly reported |
| Effect sizes | Unchanged |
| p-values | Exact match |

### 3. Meaning Preservation

| Aspect | Check |
|--------|-------|
| Core argument | Intact |
| Causal claims | Preserved |
| Hedging | Appropriate |
| Conclusions | Unchanged |

## Verification Process

```
Original Text
    ↓
Humanized Text
    ↓
┌─────────────────────┐
│ Citation Check      │ → Pass/Fail
│ Statistics Check    │ → Pass/Fail
│ Meaning Check       │ → Pass/Fail
│ Tone Check          │ → Pass/Fail
└─────────────────────┘
    ↓
Verification Report
```

## Error Categories

| Severity | Type |
|----------|------|
| **Critical** | Citation/statistic error |
| **Major** | Meaning distortion |
| **Minor** | Style inconsistency |

## Human Checkpoint Protocol

CHECKPOINT OPTIONAL (CP_HUMANIZATION_VERIFY)

Generates verification report.

## Output

- Verification checklist
- Error list (if any)
- Confidence score
- Approval recommendation
