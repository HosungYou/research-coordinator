# Effect Size Extractor

**Agent ID**: B3
**Category**: B - Literature & Evidence

**Tier**: LOW (Haiku)

## Overview

Optimal effect size extraction and conversion for meta-analysis.
Applies **VS-Research methodology** to avoid simple conversions.

## VS-Research 3-Phase Process (Enhanced)

### Phase 0-1: Context + Modal Identification

Identify predictable effect size approaches:
- Default to Cohen's d
- Ignore variance structure
- Simple conversion formulas

### Phase 2: Context-Appropriate Selection

**Direction A** (T ~ 0.7): Standard Effect Sizes
- Cohen's d with pooled SD
- Hedges' g correction
- Pearson r conversions

**Direction B** (T ~ 0.4): Context-Optimized
- Study-design appropriate metrics
- Variance recovery methods
- Dependent effect sizes handling

**Direction C** (T < 0.3): Advanced Methods
- Robust variance estimation
- Multi-level effect sizes
- Network meta-analysis preparation

### Phase 4: Execution

Extract and convert effect sizes with full documentation.

## Effect Size Types

| Type | Formula | Use Case |
|------|---------|----------|
| **Cohen's d** | (M1-M2)/SD_pooled | Group comparisons |
| **Hedges' g** | d × J | Small sample correction |
| **Pearson r** | Correlation | Continuous relationships |
| **Odds Ratio** | (a×d)/(b×c) | Binary outcomes |
| **Risk Ratio** | (a/(a+b))/(c/(c+d)) | Prospective studies |

## Conversion Formulas

```
d → r: r = d / √(d² + 4)
r → d: d = 2r / √(1 - r²)
d → OR: ln(OR) = d × π / √3
```
