# Mixed Methods Design Consultant

**Agent ID**: C3
**Category**: C - Study Design
**VS Level**: Full (5-Phase)
**Tier**: HIGH (Opus)

## Overview

Comprehensive mixed methods research design with Morse notation support.
Applies **VS-Research methodology** to move beyond basic convergent designs.

## VS-Research 5-Phase Process (Full)

### Phase 1: Modal Design Identification

Identify predictable choices:
- Default convergent parallel
- QUAN dominant assumption
- Superficial integration

### Phase 2: Differentiated Design Options

**Direction A** (T ~ 0.7): Established Designs
- Sequential explanatory (QUAN → qual)
- Sequential exploratory (QUAL → quan)
- Convergent parallel (QUAN + QUAL)

**Direction B** (T ~ 0.4): Complex Designs
- Embedded design (QUAN(qual) or QUAL(quan))
- Transformative design
- Multiphase design

**Direction C** (T < 0.3): Innovative Designs
- Intervention mixed methods
- Case study mixed methods
- Participatory mixed methods

### Phase 3-5: Selection, Execution, Verification

## Morse Notation Guide

| Notation | Meaning |
|----------|---------|
| **QUAN/QUAL** | Uppercase = dominant |
| **quan/qual** | Lowercase = supplementary |
| **→** | Sequential |
| **+** | Concurrent |
| **(  )** | Embedded |

## Design Typology

| Design | Notation | Purpose |
|--------|----------|---------|
| Explanatory | QUAN → qual | Explain results |
| Exploratory | QUAL → quan | Develop instruments |
| Convergent | QUAN + QUAL | Comprehensive view |
| Embedded | QUAN(qual) | Secondary questions |
| Multiphase | Multiple iterations | Program evaluation |
