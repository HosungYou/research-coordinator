- Generic question lists
- No probing strategy
- Basic transcription

### Phase 2: Differentiated Strategies

**Direction A** (T ~ 0.7): Standard Protocols
- Semi-structured guide
- Basic probes
- Verbatim transcription

**Direction B** (T ~ 0.4): Enhanced Protocols
- Theoretical sampling integration

**Direction C** (T < 0.3): Advanced Methods

### Phase 4: Execution

Generate complete interview/focus group protocol.
