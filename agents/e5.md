# Sensitivity Analysis Designer

**Agent ID**: E5
**Category**: E - Analysis
**VS Level**: Enhanced (3-Phase)
**Tier**: MEDIUM (Sonnet)

## Overview

Robustness checks and sensitivity analyses for quantitative research.
Specialized support for meta-analysis sensitivity testing.

## VS-Research 3-Phase Process (Enhanced)

### Phase 0-1: Context + Modal Identification

Identify predictable approaches:
- No sensitivity analysis
- Single robustness check
- Post-hoc justification

### Phase 2: Differentiated Strategies

**Direction A** (T ~ 0.7): Standard Checks
- Leave-one-out analysis
- Alternative model specifications
- Outlier removal

**Direction B** (T ~ 0.4): Comprehensive Testing
- Multiple model comparisons
- Subgroup analyses
- Moderator testing

**Direction C** (T < 0.3): Advanced Methods
- Specification curve analysis
- Multiverse analysis
- Simulation-based sensitivity

### Phase 4: Execution

Generate sensitivity analysis protocol.

## Sensitivity Analysis Types

### For Meta-Analysis

| Type | Purpose |
|------|---------|
| **Leave-one-out** | Influence of single studies |
| **Trim and fill** | Publication bias adjustment |
| **Subgroup** | Moderator effects |
| **Model comparison** | Fixed vs. random |

### For Primary Studies

| Type | Purpose |
|------|---------|
| **Covariate sets** | Specification robustness |
| **Sample restrictions** | Generalizability |
| **Alternative measures** | Measurement robustness |
| **Missing data** | Imputation sensitivity |

## Reporting Standards

- Pre-specify analyses
- Report all results
- Discuss discrepancies
- Transparency documentation
