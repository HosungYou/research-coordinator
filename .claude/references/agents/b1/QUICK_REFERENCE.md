## Database Recommendations by Type

**Systematic**: PubMed + Scopus + WoS + PsycINFO + ERIC (5+ databases)  
**Scoping**: 2-3 major databases + Google Scholar  
**Meta-Synthesis**: PsycINFO + CINAHL + Sociological Abstracts  
**Realist**: Snowballing from key papers + theory databases  
**Narrative**: 1-2 selective + key journals  
**Rapid**: PubMed + 1 discipline database (last 5 years)  

---

## Reporting Standards

- **Systematic**: PRISMA 2020 (27 items)
- **Scoping**: PRISMA-ScR (22 items)
- **Meta-Synthesis**: ENTREQ (21 items)
- **Realist**: RAMESES (24 items)
- **Narrative**: No formal standard
- **Rapid**: PRISMA-RR (adapted)

## VS 5-Phase Process

**Phase 0**: Collect review type, research question, key concepts  
**Phase 1**: Identify modal (incomplete) strategies for this review type  
**Phase 2**: Present A/B/C strategies (T=0.6/0.4/0.2)  
**Phase 3**: Select optimal strategy based on rigor needs  
**Phase 4**: Execute with detailed search strings + PRISMA diagram  
**Phase 5**: Verify comprehensiveness + checklist compliance

## Common Pitfalls by Review Type

**Systematic**: Single database, no citation tracking  
**Scoping**: Too narrow scope defeats purpose  
**Meta-Synthesis**: Using quantitative databases  
**Realist**: Linear search (should be iterative)  
**Narrative**: Cherry-picking without structure  
**Rapid**: Too comprehensive (defeats rapid purpose)
